# Customizing AtmosTrack for Your Needs

AtmosTrack lets you personalize your experience to work exactly the way you need it. This guide covers the settings you can change to make your daily work more efficient.

## Accessing Your Settings

**How to Get There:**
1. Click on your name or avatar in the top navigation bar
2. Select "Settings" from the dropdown menu
3. You'll see several tabs for different types of settings

## Personal Information Settings

### Your Profile

**What You Can Update:**
- **Your Display Name**: How your name appears to other users
- **Language Preference**: Choose your preferred language
- **Identity Information**: View your employee ID and hospital details

**How to Change Your Name:**
1. Go to the "General" tab in Settings
2. Click the edit button next to your name
3. Type your preferred display name
4. Click "Save" to confirm the change

### Language Selection

**Available Languages:**
- **English**: Default international language
- **Suomi**: For Finnish hospitals and staff
- **Eesti**: For Estonian healthcare facilities  
- **العربية (Arabic)**: With right-to-left text support

**How to Change Language:**
1. In Settings, find the "Language" dropdown
2. Select your preferred language
3. The interface updates immediately
4. Your choice is saved automatically for next time

**Language Benefits:**
- All menus and buttons appear in your chosen language
- Date and time formats match your region
- Number formatting follows local conventions
- Help text and messages appear in your language

## Software Updates

### Understanding Updates

**What Updates Include:**
- New features to make your work easier
- Security improvements for data protection
- Bug fixes for smoother operation
- Performance improvements for faster scanning

**Update Channels:**
- **Stable**: Thoroughly tested updates (recommended for most users)
- **Beta**: Early access to new features (for advanced users)

### Managing Updates

**Current Version Information:**
- See what version of AtmosTrack you're using
- Check when it was last updated
- View your current update channel (Stable or Beta)

**Checking for Updates:**
1. Go to the "Updates" tab in Settings
2. Click "Check for Updates" button
3. System will search for new versions
4. Follow prompts to install if updates are available

**Choosing Your Update Channel:**
- **Choose Stable If**: You want reliable, tested software
- **Choose Beta If**: You want to try new features early and don't mind occasional issues

## Printer and Label Settings

### Setting Up Your Printer

**Initial Printer Setup:**
1. Go to the Debug page (for administrators)
2. Find the "Label Printing" section
3. Click "Refresh" to find available printers
4. Select your barcode label printer from the list

**Recommended Printers:**
- Intermec PF8T for standard hospital labels
- Any printer that supports 4cm x 6cm labels
- Thermal transfer printers work best

### Label Printing Options

**Basic Settings:**
- **Employee Name**: Your name appears on printed labels
- **Date Format**: Automatic date stamping
- **Barcode Type**: Code 128 format for maximum compatibility

**Advanced Options (for supervisors):**
- **Print Quality**: Adjust darkness and clarity
- **Label Size**: Confirm 4cm x 6cm dimensions
- **Orientation**: Portrait mode for standard labels

## Troubleshooting and Support

### Debug Tools (For Technical Issues)

**When to Use Debug Tools:**
- Login problems with barcode or Microsoft account
- Printer not working correctly
- Software running slowly
- Scanner not responding

**Basic Troubleshooting:**
1. Go to Settings → Debug tab
2. Try "Refresh Token" if login issues occur
3. Use "Clear Cache" if system seems slow
4. Test printer functionality if labels won't print

**When to Contact Support:**
- Debug tools don't solve the problem
- Error messages you don't understand
- Equipment hardware issues
- Need help with new features

### Getting Help

**Internal Support:**
- Ask your shift supervisor for policy questions
- Contact your hospital's IT department for technical issues
- Check with the CSSD manager for workflow questions

**System Support:**
- Built-in help tooltips throughout the interface
- Step-by-step guides for common tasks
- Error messages with helpful suggestions
- Documentation and training materials

## Backup and Recovery

### Settings Backup

**What Gets Saved:**
- Your language preferences
- Notification settings
- Printer configurations
- Personal customizations

**Automatic Saving:**
- Settings are saved automatically when changed
- Preferences persist across browser sessions
- Settings sync across devices when you log in
- No manual backup required for personal settings

### Recovery Options

**If Settings Are Lost:**
- Log out and log back in to restore settings
- Contact IT support if preferences don't restore
- Reconfigure printers if hardware settings are lost
- Re-select language if interface appears in wrong language
